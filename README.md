# Express + Next.js Template

Fullstack template gồm:

* Express (TypeScript)
* Next.js App Router
* Redux Toolkit
* Ant Design
* next-intl

## Structure

```txt
templates/
  fullstack/
    server/
    client/
```

## Backend

```bash
cd templates/fullstack/server
npm install
npm run dev
```

Server chạy tại:

```txt
http://localhost:9000
```

## Frontend

```bash
cd templates/fullstack/client
npm install
npm run dev
```

Frontend chạy tại:

```txt
http://localhost:3000
```
